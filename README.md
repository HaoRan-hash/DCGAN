# 项目简介 :clipboard:
该项目是利用DCGAN生成你想要的图片，我使用了3个图集进行训练，分别是鲜花、celeba人脸、动漫头像。  
项目一共有三个py文件，功能如下：network.py是实现DCGAN的神经网络结构，train.py是训练过程，generate.py是利用训练好的模型参数生成图片。  
每一个py文件中都有详细的注释说明。
